Cybersecurity Journey

I am a Computer Science Engineering graduate and cybersecurity enthusiast with hands-on experience in SOC operations, vulnerability assessment, penetration testing, and security monitoring. My interest in cybersecurity began with learning networking fundamentals, Linux systems, and web technologies, which gradually led me toward exploring both offensive and defensive security practices.

To strengthen my practical knowledge, I actively practice cybersecurity through hands-on labs and real-world simulations on TryHackMe. Through this platform I have completed more than 26 security labs and achieved a ranking within the Top 15%, where I practiced areas such as penetration testing, network scanning, web application security, and privilege escalation.

I also completed a three-month Cybersecurity Internship at Unified Mentor Pvt. Ltd., where I worked on projects related to Windows services and SentinelShield security monitoring. During this internship I gained practical exposure to network security concepts, vulnerability assessment, threat analysis, and basic incident response procedures used in security operations.

My learning journey also includes practical experience in Security Operations Center (SOC) activities such as log monitoring, threat detection, and security alert analysis. I worked with SIEM tools including Wazuh and Splunk to understand how security events are monitored, analyzed, and investigated within enterprise environments.

In addition to defensive security, I am actively exploring ethical hacking and vulnerability research through bug bounty programs on platforms like HackerOne, Bugcrowd, and Intigriti. These platforms allow security researchers to legally test applications for vulnerabilities and contribute to improving the security of real-world systems.

I have also completed training for the Certified Ethical Hacker (CEH) program and earned the Certified SOC Analyst (CSA) certification , which validates my knowledge in SOC operations, threat detection, and security monitoring. 

Along with offensive security practice, I continue to build defensive security skills through blue team platforms such as LetsDefend and Blue Team Labs Online, where I practice incident analysis, log investigation, and threat detection scenarios.

My long-term goal is to develop strong expertise in both offensive and defensive cybersecurity, focusing on SOC analysis, threat hunting, vulnerability research, and ethical hacking while continuously improving my practical skills through labs, research, and real-world security challenges.
