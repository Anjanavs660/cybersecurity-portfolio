# Security Tools Practice
During my cybersecurity training and lab practice, I gained hands-on experience with several important security tools used in penetration testing and security operations.

## Nmap
Nmap is a network scanning tool used to discover hosts, identify open ports, and detect services running on a network. It helps security professionals understand the attack surface of a system and identify potential vulnerabilities.

## Burp Suite
Burp Suite is a web application security testing tool used to intercept and analyze HTTP/HTTPS requests between the browser and web server. It is commonly used for testing web vulnerabilities such as SQL injection, authentication issues, and other application flaws.

## Wireshark
Wireshark is a packet analysis tool used to capture and inspect network traffic in real time. It helps in analyzing network communication, detecting suspicious activity, and troubleshooting network-related security issues.

## Metasploit
Metasploit is a penetration testing framework used to identify, test, and exploit vulnerabilities in systems. It provides a large collection of exploits and payloads that help security researchers simulate real-world attacks in a controlled environment.

## Splunk
Splunk is a security information and event management (SIEM) tool used for log analysis, monitoring system events, and detecting security threats within an organization’s infrastructure.

## Wazuh
Wazuh is an open-source SIEM and XDR platform used for threat detection, log monitoring, intrusion detection, and security event analysis across different systems and environments.

These tools helped me understand practical aspects of vulnerability scanning, web application testing, network monitoring, exploitation testing, and security event analysis.
